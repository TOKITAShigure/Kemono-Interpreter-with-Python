# Kemono Interpreter with Python

## Overview
プログラミング言語 Kemono https://github.com/consomme/kemono_friends_lang のPython インタプリタです。

## Usage
```
python kf2py.py kaban_chang.kemono
```

入力ファイルと実行し、ポインタ数の上限を入力すると結果を出力します。
